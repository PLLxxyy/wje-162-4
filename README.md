# 居民垃圾分类积分打卡系统

全栈居民垃圾分类积分打卡系统，居民每天打卡记录垃圾分类情况获取积分，连续打卡有额外奖励，积分可在商城兑换生活用品。

## 技术栈

- **前端**: Vite + React 18 + TypeScript，默认端口 5182
- **后端**: Express + TypeScript + better-sqlite3，默认端口 3212
- **认证**: JWT + bcryptjs
- **启动**: concurrently 同时启动前后端

## 启动方式

```bash
# 安装所有依赖（根目录、server、client）
npm run install:all

# 同时启动前后端
npm run dev
```

启动后访问 http://localhost:5182

## 功能说明

### 居民端
- **登录注册**: 账号密码登录，支持新用户注册
- **每日打卡**: 选择垃圾分类类型（可回收物/厨余垃圾/有害垃圾/其他垃圾）、填写投放重量、选择投放点，提交获得积分
- **打卡日历**: 日历视图展示每月打卡记录，一目了然
- **连续打卡奖励**: 连续2天+2分，连续3天+5分，连续7天+15分
- **积分排行榜**: 本月积分排名前50名居民，显示连续打卡天数
- **积分商城**: 用积分兑换生活用品（垃圾袋、分类桶、洗手液等）
- **个人中心**: 打卡记录、积分明细、兑换记录、社区公告
- **公告轮播**: 首页展示最新社区公告

### 管理员端
- **今日统计**: 打卡人数、分类投放量、积分发放总量
- **月度趋势**: 每日打卡人数柱状图、分类投放量统计
- **公告管理**: 发布和删除社区公告

## 积分规则

| 垃圾类型 | 基础积分 |
|----------|----------|
| 可回收物 ♻️ | 10 分 |
| 有害垃圾 ⚠️ | 8 分 |
| 厨余垃圾 🥬 | 5 分 |
| 其他垃圾 🗑️ | 3 分 |

连续打卡额外奖励：2天+2分，3天+5分，7天+15分

## 注意事项

- 首次启动会自动建表并插入测试数据（含42位居民用户和10款商品）
- 数据库文件存储在 `server/data.db`，删除可重置所有数据
- 前端通过 Vite 代理 `/api` 请求到后端 3212 端口
- 可通过 `VITE_PORT`、`VITE_API_PORT`、`PORT` 和 `JWT_SECRET` 覆盖默认配置
- 每天只能打卡一次，重复打卡会提示
- 所有样式写在 `index.html` 的 `<style>` 标签中，无第三方 UI 库
